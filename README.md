# Generador de Informes Técnicos - GEMSA S.A.C.

Esta es una aplicación web de alta gama diseñada para agilizar y estandarizar la creación de informes técnicos de visitas de campo para **GEMSA Corporación S.A.C.**, líder en servicios generales y mantenimiento de infraestructura en Lima, Perú.

## Características de la Aplicación

- **Estética Visual Premium:** Diseño de "Glassmorphic Dark Mode" optimizado para ofrecer una interfaz limpia, corporativa y futurista, con fuentes estilizadas de Google Fonts (*Outfit*).
- **Vista Previa en Tiempo Real:** Visualización en vivo de la salida de texto estructurado de acuerdo con la plantilla oficial del informe, lista para su inserción directa.
- **Flujo de Trabajo Dinámico:**
  - Formulario estructurado para ingresar datos del cliente, códigos correlativos 2026, descripción técnica detallada del trabajo y fecha.
  - Creador y gestor dinámico de reporte fotográfico (con previsualizaciones de fotos, descripciones técnicas y estampas de fecha/hora individuales).
  - Creador y gestor de conclusiones y recomendaciones numeradas ilimitadas.
- **Acciones Rápidas Integradas:**
  - **Copiar Informe:** Copia directa al portapapeles con micro-animaciones interactivas de éxito.
  - **Descargar TXT:** Descarga directa a su ordenador en formato de texto estructurado codificado en UTF-8 con nombre de archivo inteligente (`[Código]_[Cliente].txt`).
  - **Autogenerador de Códigos:** Generación rápida de códigos correlativos únicos para el año 2026 (`IT-2026-XXXX`).
  - **Limpieza Completa:** Botón para reiniciar todos los campos del formulario de forma segura.
- **Diseño Responsivo:** Completamente adaptable a pantallas móviles, tabletas y ordenadores de escritorio.

## Tecnologías Utilizadas

- **HTML5:** Marcación semántica y accesible para toda la estructura de la aplicación.
- **Vanilla CSS3:** Maquetación avanzada utilizando CSS Grid, Flexbox, variables personalizadas para diseño modular, efectos de desenfoque de fondo (*backdrop-filter*) y transiciones fluidas.
- **JavaScript (ES6+):** Gestión interactiva de estado local de imágenes, renderizado dinámico en el DOM, API de Clipboard para copiado rápido y generación dinámica de archivos Blob para la descarga directa.

## ¿Cómo abrir y utilizar la aplicación localmente?

No requiere de servidores complejos ni dependencias externas pesadas. Simplemente:

1. Clonar este repositorio localmente en su máquina.
2. Hacer doble clic en el archivo **[index.html](index.html)** para abrirlo directamente en cualquier navegador web moderno (Google Chrome, Microsoft Edge, Mozilla Firefox o Safari).
3. Rellenar los campos requeridos en el panel izquierdo y ver los cambios reflejados automáticamente en el panel derecho.
4. Presionar el botón de **Copiar Informe** para pegar el contenido técnico directamente en su procesador de textos o presionar **Descargar TXT** para archivar una copia digital.
